# Paid Tool Exports Needed Later

Policy: do not buy paid SEO tools yet. Use these exports only if Phase 1 and Phase 2 show promise.

Prefer CSV exports over login access.

## Ahrefs

Official feature notes checked: Ahrefs Keywords Explorer describes keyword ideas, keyword difficulty, search volume, traffic potential, parent topic, SERP review, historical rankings, and weak-site filters.

### What Question It Helps Answer

- Which competitor pages get organic traffic?
- Which keywords do those pages rank for?
- How hard are target keywords based on backlinks to ranking pages?
- Are there parent topics that should be targeted instead of fragmented long tails?
- Which domains repeatedly rank in the niche?

### When We Should Buy/Use It

Later, only after one niche is likely selected and we are preparing 20-30 pages.

Use it for one month if:

- Manual SERP validation shows clear opportunity.
- We need competitor top-pages and content-gap data.
- We want to avoid publishing pages that should be consolidated under broader parent topics.

### CSV Exports To Provide

- Keywords Explorer export for 100-300 seed queries:
  - keyword
  - country
  - volume
  - keyword difficulty
  - traffic potential
  - parent topic
  - clicks/clicks per search if available
  - SERP overview metrics
- SERP overview export for top 50 candidate keywords:
  - ranking URL
  - domain
  - title
  - backlinks/referring domains
  - estimated traffic
  - domain rating
- Site Explorer export for 5-10 competitor domains:
  - top pages
  - organic keywords per page
  - estimated traffic
  - referring domains
- Content gap export comparing selected competitors.
- "Also ranks for" / organic keywords export for competitor pages if available.

### Decision Supported

- Which 20-30 pages to publish first.
- Whether to merge long-tail pages into broader guides.
- Which competitors are beatable.
- Whether backlinks are likely required.

### Necessary Now?

Later. Not necessary for Phase 1.

## Semrush

Official feature notes checked: Semrush positions itself around SEO plus AI visibility, with keyword, backlink, traffic, content, local, and AI visibility workflows.

### What Question It Helps Answer

- What keyword clusters and gaps exist across competitors?
- Which competitors own the broad topic?
- Which keywords have informational, commercial, local, or navigational intent?
- How might the site appear in classic search and AI/search visibility workflows later?

### When We Should Buy/Use It

Later, only if one niche is selected and we need broader workflow support than Ahrefs, or if we want keyword-gap and AI visibility features in the same month.

Use it for one month if:

- We are publishing 20-30 pages.
- We want a broad keyword gap analysis across several domains.
- We may use rank tracking after launch.

### CSV Exports To Provide

- Keyword Magic Tool export:
  - keyword
  - intent
  - volume
  - keyword difficulty
  - trend
  - CPC
  - competitive density
  - SERP features
- Keyword Gap export for 5-10 competitor domains:
  - missing keywords
  - weak keywords
  - competitor rankings
  - volume
  - KD
  - URL if available
- Organic Research export for competitor domains:
  - top pages
  - organic keywords
  - estimated traffic
  - position
  - URL
- SERP Gap Analyzer or similar export if available.
- Position tracking export after launch:
  - keyword
  - URL
  - position
  - change
  - SERP features

### Decision Supported

- Which niche has the strongest keyword gap.
- Which first pages map to real query clusters.
- Which competitors matter most.
- What to monitor after launch.

### Necessary Now?

Later or optional. Not necessary for Phase 1.

## LowFruits

Official feature notes checked: LowFruits focuses on bulk SERP analysis, weak spots, long-tail keyword ideas, clustering, and low-authority sites ranking in SERPs.

### What Question It Helps Answer

- Which long-tail keywords have weak SERPs?
- Where are low-authority sites ranking?
- Which keyword clusters share intent?
- Which phrases should be prioritized for a new small site?

### When We Should Buy/Use It

Potentially first paid tool, but only if two niches score 75+ or manual SERP review finds many possible weak-result queries.

Use it before Ahrefs/Semrush if:

- We mainly need low-competition long-tail validation.
- We have 100-300 candidate keywords from Phase 1.
- We want to choose between two promising niches cheaply.

### CSV Exports To Provide

- Keyword analysis export:
  - keyword
  - volume if available
  - weak spots count
  - low DA/low authority ranking pages
  - allintitle/KGR if available
  - SERP difficulty indicators
  - cluster
  - intent
- SERP details export:
  - ranking URL
  - domain
  - title
  - DA/authority metric
  - word count/title relevance if available
- Cluster export:
  - cluster name
  - included keywords
  - shared intent
  - best target page

### Decision Supported

- Which niche is easiest for a new site to enter.
- Which first 20 pages should be prioritized.
- Which queries need manual review before writing.

### Necessary Now?

Optional soon if Phase 2 keeps two niches above 75.

## KeySearch

### What Question It Helps Answer

- What are budget-friendly search volumes and difficulty estimates?
- Which keywords can be tracked after launch?
- Which competitor URLs are ranking for the target query set?

### When We Should Buy/Use It

Optional budget alternative to LowFruits or Semrush/Ahrefs.

Use it if:

- We want cheaper keyword research and rank tracking.
- We are not ready to pay for Ahrefs/Semrush.
- We need a sanity check on query volume and difficulty.

### CSV Exports To Provide

- Keyword research export:
  - keyword
  - volume
  - difficulty
  - CPC
  - PPC/competition if available
  - trends if available
- SERP analysis export:
  - top ranking URLs
  - domain authority/page authority metrics if available
  - backlinks if available
  - title
- Rank tracking export after launch:
  - keyword
  - URL
  - position
  - change
  - date

### Decision Supported

- Whether the niche has enough low-difficulty long tails.
- Which pages to publish first.
- Which post-launch rankings are moving.

### Necessary Now?

Optional. Consider only after Phase 2.

## Screaming Frog

Official feature notes checked: Screaming Frog SEO Spider audits technical SEO issues, crawls 500 URLs free, exports onsite elements, finds broken links, audits metadata, integrates with GSC/GA/PSI in paid mode, and visualizes site structure.

### What Question It Helps Answer

- Are published pages crawlable and indexable?
- Are titles, meta descriptions, H1s, canonicals, internal links, and status codes correct?
- Are there broken links, redirects, duplicates, thin pages, or technical issues?
- Does the site architecture support SEO?

### When We Should Buy/Use It

Later, after publishing.

Use the free version first if the site has fewer than 500 URLs. Consider paid only when the site grows or needs scheduled/API-integrated audits.

### CSV Exports To Provide

- Internal HTML export:
  - address
  - status code
  - indexability
  - title
  - title length
  - meta description
  - H1
  - word count
  - crawl depth
  - inlinks
  - outlinks
  - canonical
- Response codes export.
- Page titles export.
- Meta descriptions export.
- H1 export.
- Inlinks export for internal linking analysis.
- GSC/GA integrated exports later if paid and connected.

### Decision Supported

- What technical fixes are needed after launch.
- Which pages need metadata or internal-link improvements.
- Whether the internal linking structure matches the launch plan.

### Necessary Now?

Later. Free crawl likely enough for initial launch.
