# Content Principles

## Parent Promise

A tired parent should be able to scan a page, gather normal household materials, and know what to say first.

## Activity Standard

Every activity should have:

- a clear parent job
- a clear kid mission
- short kid-facing steps
- concrete safety and mess boundaries
- a stop rule or rescue line
- an easier version
- a harder version when useful
- notes from real testing when available

## Originality Standard

Original does not mean every idea is invented from nothing. It means Kid Activity Lab adds useful parent value:

- simplified setup
- safer boundaries
- kid-facing wording
- a test or build loop
- real observations
- visuals or diagrams when possible
- honest notes about what failed

## SEO Standard

An indexable page should deserve a search visit. It should not exist only because a keyword exists.

Strong pages should have at least one of:

- original testing notes
- a genuinely useful chooser or decision table
- unique age/material/safety guidance
- a strong set of relevant activity cards
- photos, diagrams, or other helpful visuals
- specific parent rescue notes

For early GSC-visible pages, prefer improving the ranking or impression-earning URL before creating a new page for the same intent.

Separate evidence types:

- Measured data: GSC, analytics, live index status, real parent testing.
- Tool estimates: Semrush, DataForSEO, keyword tools, SERP estimates.
- Assumptions: likely parent intent, expected search behavior, future monetization.
- Opinions: editorial judgment, taste, and prioritization.

If volume, CPC, difficulty, traffic, backlinks, revenue, rankings, or index status are unavailable, write `UNKNOWN` rather than guessing.

## Video Policy

Reference videos may be used for inspiration or setup visualization. Always credit and link to the source.

Do not scrape, download, rehost, or imply creator endorsement.

When a page includes a reference video, Kid Activity Lab still needs to add enough original instructions, safety notes, and parent guidance to stand on its own.
